Current Projects

Tim Berners-Lee is currently a professor at MIT. Him and his students are currently working on a project called "Solid." This project is designed to improve privacy and provide true ownership over data uploaded to the internet. Its a way to protect internet users personal data and sensitive information. It goes against the social media companies scraping away any bits of privacy we have left as internet users. It is an open source program that contains a set of tools that allows users the freedom to choose where their data resides and who is allowed to access it.
